CV Personal

Currículum vitae desarrollado en HTML5 y CSS3. Incluye información personal, experiencia laboral, formación académica, competencias técnicas e idiomas.

Estructura del proyecto

El proyecto está organizado en las siguientes carpetas y archivos:

- `index.html`: Archivo principal que contiene la estructura semántica del CV
- `css/reset.css`: Reset CSS para normalizar estilos entre navegadores
- `css/style.css`: Estilos principales del proyecto con variables CSS y diseño responsive
- `assets/yo.jpg`: Imagen de perfil profesional

## Características técnicas

El CV está construido utilizando HTML5 semántico con elementos como `header`, `main`, `aside`, `section` y `article` para una estructura clara y accesible. El diseño implementa:

- **Variables CSS**: Sistema de colores centralizado mediante variables CSS personalizadas
- **Diseño responsive**: Adaptación a diferentes tamaños de pantalla mediante media queries y flexbox
- **Tipografía**: Uso de la fuente Poppins de Google Fonts con diferentes pesos (300, 400, 500, 600, 700)
- **Barras de progreso**: Visualización de niveles de idiomas mediante barras de progreso personalizadas
- **Enlaces interactivos**: Enlaces a GitHub, correo electrónico y ubicación con Google Maps
- **Botón de scroll**: Botón flotante para volver al inicio de la página

Secciones incluidas

El CV presenta las siguientes secciones organizadas en un layout de dos columnas (aside y main):

- Información de contacto con enlaces funcionales
- Formación académica y titulaciones
- Proyectos con enlace a GitHub
- Idiomas con indicadores visuales de nivel
- Resumen profesional
- Experiencia laboral detallada
- Competencias técnicas
- Competencias personales

Tecnologías utilizadas

- HTML5 semántico
- CSS3 con variables personalizadas
- Google Fonts (Poppins)
- Diseño responsive con Flexbox
- Reset CSS para compatibilidad cross-browser
