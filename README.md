# CV Personal

Este repositorio contiene mi currículum vitae en formato HTML.

## Contenido

El CV incluye las siguientes secciones:
- **Información personal**: Datos de contacto y disponibilidad
- **Idiomas**: Español, Catalán y Rumano (nativos), Inglés (nivel alto), Francés (básico) y Chino (HSK 2)
- **Resumen profesional**: Perfil y objetivos profesionales
- **Experiencia laboral**: Historial en seguridad privada
- **Formación**: Estudios actuales y titulaciones
- **Competencias técnicas**: Conocimientos en redes, sistemas operativos y desarrollo web

## Proceso de desarrollo

El CV está desarrollado con HTML5 y CSS3, utilizando una estructura semántica y diseño responsive. Se ha implementado un sistema de reset CSS para garantizar la consistencia visual entre diferentes navegadores, junto con estilos personalizados que priorizan la legibilidad y presentación profesional.
